# Number Guess Game 🎯

A simple Number Guess Game built using HTML, CSS, and JavaScript.

## 📌 Features

- User sets the maximum limit
- Random number generation
- Gives hints:
  - Too High
  - Too Low
  - Correct Guess
- Simple and clean UI
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🚀 How to Run

1. Download or clone the project
2. Open the project folder
3. Run `index.html` in your browser

## 📷 Project Preview

The user enters:
- Maximum limit
- Guess number

The game checks whether the guess is:
- Correct
- Too High
- Too Low
